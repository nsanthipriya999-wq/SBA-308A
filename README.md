#NASA ASTRONOMY SPACE EXPLORER


An interactive web application that uses NASA's Astronomy Picture of the day (APOD) API to display stunning space image on the particular selected date with name and description. 

#How to use
1.Select any date to view APOD 
2.Click on "Explore APOD" button
3.View Astronomy Picture of the day for the selected date.
4.Read the detailed description provided by NASA.

#Features
.Select any available APOD date(from June 16 1995 to today).
.Displays NASA's Astronomy Picture Of the Day.
.Stylish space-themed interface.
.Smooth User Interface and responsive design.
.Animated loading progress bar.
.Colourful Space images
.Space themed design with modern styling.
.Detailed Explanation for every APOD.
.Supports both images and videos.
